T-Rex

T-Rex (Theory Research Exchange) aims to improve the process of literature reviewing and knowledge synthesis. T-Rex supports digitally encoding publications' core knowledge contributions and ancillary data via a labeled property graph database. Encoding knowledge into a graph database brings new opportunities to manage and analyze knowledge. There is an application to facilitate the coding process - Codasaurus. Codasaurus automatically generates Cypher code for a graph version of a publication with information input by a researcher. Analysaurus is under development to support querying and analytics.
